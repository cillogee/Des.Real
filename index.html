EquipDrop

A smart PPE redistribution system designed for students, workshops and practical learning spaces.

EquipDrop allows usable PPE to be stored, managed and accessed through a controlled locker system.

About

EquipDrop provides affordable and accessible PPE to students while supporting more responsible use of resources. The platform is designed to reduce unnecessary waste by redistributing PPE that is still usable within an educational setting.

The system combines a digital ordering page with a physical locker unit. Once PPE is selected, the user can collect it from a controlled locker compartment using an access code.

How It Works

  1. Select the PPE item you need from the catalogue.
  2. Choose the correct size and quantity.
  3. Submit your order through the webpage.
  4. An access code is assigned to the correct locker compartment.
  5. Enter the code on the keypad to open the locker and collect your PPE.

Sustainability

This project supports SDG 12: Responsible Consumption and Production. By redistributing PPE instead of allowing it to become waste, EquipDrop helps extend the lifecycle of materials and supports circular economy thinking.

The system encourages students to take only what they need, reuse available resources and think more carefully about waste in practical learning environments.

Available PPE

  • Hard hats
  • Safety glasses
  • Safety jackets
  • Protective trousers
  • Safety boots
